# Data Storage

Folder ini merupakan penyimpanan sementara atau penyimpanan statis untuk kebutuhan dataset. Biasanya, direktori ini berisi salinan dataset mentah berformat `.csv` (misal `aquaculture_water_quality.csv` atau data turunan bersihnya).

Pada arsitektur sistem terbaru ini, dataset utama telah dipindahkan secara permanen dan dikelola secara sentral melalui basis data Supabase (tabel `water_quality_clean`).
