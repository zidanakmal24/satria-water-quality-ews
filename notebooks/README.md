# Jupyter Notebooks (Eksperimen Data & Prototyping)

Direktori ini memuat seluruh berkas eksperimental berekstensi `.ipynb`. Segala tahapan riset (*R&D*) sebelum kode direfaktorasi (*refactoring*) ke dalam skrip *production* Microservice akan dilakukan di sini.

Buku catatan ini memuat proses eksplorasi mendalam (Exploratory Data Analysis / EDA awal), pembersihan anomali (*Outliers*), imputasi data kosong, seleksi algoritma (PyCaret), hingga draf pertama pembuatan rancangan *Machine Learning Operations* (MLOps). Sangat berguna bagi akademisi atau rekan tim yang ingin melihat "coretan dasar" dibalik performa dan keputusan arsitektur ML.
